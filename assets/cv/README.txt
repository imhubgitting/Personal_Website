CV Placeholder
===============

To add your CV to the website:

1. Place your CV file (preferably PDF format) in this folder
2. Rename it to something simple like "Gavin_Antonacci_CV.pdf" or just "cv.pdf"
3. Update the link in index.html (line with href="assets/cv/placeholder.pdf")
   Change it to: href="assets/cv/YOUR_CV_FILENAME.pdf"

Example:
<a href="assets/cv/Gavin_Antonacci_CV.pdf" download class="cv-button">

Current placeholder file: This README.txt file serves as a placeholder
Replace this file with your actual CV PDF.
